# Test Calculator

A small browser calculator built with plain HTML, CSS, and JavaScript while testing calculator work with AI tools.

## Features

- Addition, subtraction, multiplication, and division
- Decimal input
- Clear and delete controls
- Keyboard support
- No build step or dependencies

## How to Run

Open `index.html` in a browser.

## Files

- `index.html` - calculator UI, styles, and logic
- `scripts/auto-commit.sh` - local helper for automatic Git commits
